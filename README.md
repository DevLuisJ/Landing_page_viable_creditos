# Viable Créditos - Landing Page

Landing page optimizada para la conversión de leads y gestión de productos financieros (Financiación vehicular, Compra de cartera y Seguros). Este proyecto está diseñado con un enfoque en **Growth Marketing** y **Estrategia Multibanca**.

## 🚀 Características Principales

### 1. Sección Hero de Alto Impacto
- **Propuesta de Valor Clara**: "Tu crédito de vehículo a la velocidad de un clic".
- **Optimización Visual**: Overlay de alto contraste para máxima legibilidad.
- **Micro-copy de Urgencia**: Aclaración sobre el proceso digital y gratuito para reducir la fricción.
- **Trust Elements**: Badge de seguridad para generar confianza inmediata.

### 2. Estrategia Multibanca (17 vs. 1)
- **Sección de Beneficios**: Resalta la ventaja competitiva de contar con convenios con 17 entidades financieras.
- **Diseño Ergonómico**: Tarjetas interactivas con efectos hover y alineación perfecta mediante Flexbox.
- **Copy Persuasivo**: Enfoque en ahorro de tiempo, mejores tasas y procesos sin papeleo.

### 3. Portafolio de Productos
- **Crédito Vehicular**: Asesoría personalizada y respuesta rápida.
- **Compra de Cartera**: Estructura detallada con beneficios específicos (Baja de tasa, cuotas pequeñas, liquidez).
- **Póliza Todo Riesgo**: Cobertura integral con las mejores aseguradoras.
- **Conversión Directa**: Botones de WhatsApp integrados en cada tarjeta para contacto inmediato.

### 4. Experiencia de Usuario (UX/UI)
- **Navegación Inteligente**: Scroll suave con márgenes ajustados para que las secciones aparezcan centradas.
- **Layout Responsivo**: Diseño adaptable a móviles, tablets y escritorio.
- **Estética Premium**: Uso de tipografías modernas (DM Sans, Poppins) y una paleta de colores profesional.

## 🛠️ Tecnologías Utilizadas
- **HTML5**: Estructura semántica completa.
- **CSS3**: Variables CSS (Custom Properties), Flexbox, Grid Layout y efectos de cristalmorfismo.
- **JavaScript**: Lógica para navegación y elementos dinámicos.
- **SVG**: Iconografía vectorial para tiempos de carga ultrarrápidos.

## 📦 Estructura del Proyecto
- `index.html`: Punto de entrada principal.
- `css/styles.css`: Hoja de estilos principal con el sistema de diseño.
- `js/main.js`: Lógica de interactividad.
- `assets/`: Imágenes de alta calidad y logos.

## 🌐 Despliegue
El sitio está configurado para desplegarse mediante **GitHub Pages** con soporte HTTPS para conexiones seguras.
